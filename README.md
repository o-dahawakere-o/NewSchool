Данный проект является неофициальным форком и не имеет никакого отношения к авторам оригинального клиента NetSchool.

На свой страх и риск: Автор (DaHawakere) не несет ответственности за любые последствия использования данного приложения. Если ваш телефон решит устроить косплей сверхновой, аккаунт заблокируют, или оценки внезапно превратятся в тыкву — это ваша зона ответственности.

Конфиденциальность: Приложение работает локально. Ваши токены, пароли и прочие данные авторизации отправляются только на официальные серверы NetSchool и/или сохраняются локально. Никаких «облаков», промежуточных серверов, прокси, или кражи данных — я ценю приватность не меньше вашего.

Чистота кода: Весь исходный код открыт. Вы можете лично убедиться в отсутствии «закладок».
